## Directory Instructions

### 1. APKPure (DR 87) — HIGHEST PRIORITY

**URL:** https://developer.apkpure.com/

**Steps:**
1. Register account at https://developer.apkpure.com/
2. Click "Upload App"
3. For each app:
   - Upload `app-release.aab` (APKPure accepts AAB)
   - Fill metadata from `metadata.json`
   - Category: **Lifestyle** → subcategory close to **Esoteric/Magic**
   - Add tags from metadata
   - Link website: `https://cha0smagicklabs.com/apps/[ID].html`
   - Link privacy: `https://cha0smagicklabs.blogspot.com/`
4. Submit → listing generates backlink: `https://apkpure.com/[app-name]/[package-name]`

### 2. Uptodown (DR 84)

**URL:** https://www.uptodown.com/developers

**Steps:**
1. Register at https://www.uptodown.com/developers
2. Click "Add App"
3. For each app:
   - Upload `app-debug.apk` (Uptodown prefers APK)
   - Title, description from `metadata.json`
   - Category: **Lifestyle**
   - Price: **Paid** (not free)
   - Website URL from metadata
4. Submit → listing at: `https://[app-name].en.uptodown.com/android`

### 3. Softonic (DR 80)

**URL:** https://developer.softonic.com/

**Steps:**
1. Register at https://developer.softonic.com/
2. Create new app listing
3. For each app:
   - Upload `app-release.aab` or `app-debug.apk`
   - English description from `metadata.json`
   - Category: **Lifestyle**
   - Set price as paid
   - Add screenshots
4. Submit for review (may take 2-5 days)

### 4. Aptoide (DR 76)

**URL:** https://developer.aptoide.com/

**Steps:**
1. Register at https://developer.aptoide.com/
2. Click "Upload App"
3. For each app:
   - Upload `app-debug.apk`
   - Metadata from `metadata.json`
   - Category: **Lifestyle**
   - Set store: choose your own store or Cha0smagick Labs store
4. Submit → listing at: `https://[app-name].en.aptoide.com`

### 5. AppBrain (DR 75) — AUTO-INDEXING

**URL:** https://www.appbrain.com/

**Steps:**
1. Register at https://www.appbrain.com/
2. This directory **auto-indexes from Google Play**, so if your apps are already on Play Store, they'll appear automatically
3. To accelerate: manually submit your Google Play URLs
4. Verify developer identity
